## Hardware components 

### 1. HC-05
- get unlock input byte from the app

### 2.RC522 RFID reader
- read rfid tags

### 3.Atmega 328p microcontroller
- verify read rfid nuid and received bluetooth bytes
- activate lock on successful bluetooth or rfid authentication
